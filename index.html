Samba

Shares and user passwords

In /etc/samba/smb.conf modify or add the following

  • server name=
  • workgroup=HOME
  • read only=no
  • Add [Plex_media] share
  • browseable=yes
  • writeable=yes
  • create mask=0777
  • dir. mask=0777

Test smb config with testparms
Add user samba password with smbpasswd -a "username "
